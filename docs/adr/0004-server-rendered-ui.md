# ADR-0004: Server-rendered UI with Django templates

## Status
Accepted

## Context
Проект ориентирован на внутренний веб-интерфейс склада с формами и CRUD-операциями. Нужна быстрая разработка и простая эксплуатация.

## Decision
Использовать серверный рендеринг страниц через Django templates и class-based/function-based views.

## Consequences
Плюсы:
- Быстрое создание CRUD-интерфейсов.
- Простая аутентификация и защита страниц средствами Django.
- Меньше фронтенд-инфраструктуры.

Минусы:
- Ограниченная интерактивность по сравнению с SPA.
- Логика представления сильнее привязана к backend.

## Alternatives Considered
### Option 1: SPA (React/Vue) + API backend
Более гибко для rich UI, но выше сложность и стоимость разработки.

### Option 2: Гибридный подход с постепенным SPA
Подходит на будущее, но не выбран как базовая стратегия текущего этапа.
